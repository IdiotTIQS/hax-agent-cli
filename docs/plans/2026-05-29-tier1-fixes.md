## Task 1: 对话压缩重写 (P0)

**来源:** OpenHarness `services/compact/__init__.py` (1727 行)

### 新增能力:
1. `microCompact(messages)` — 清除可压缩工具的旧结果，保留最近 5 个
2. `contextCollapse(messages)` — 确定性文本截断
3. `splitPreservingToolPairs(messages)` — 不拆分 tool_use/tool_result 对
4. `groupByPromptRound(messages)` — 按用户提示轮次分组
5. `getContextWindow(model)` — 按模型获取上下文窗口
6. `getAutocompactThreshold(model)` — 自动压缩阈值计算
7. `CompactionProgressEvent` — 结构化进度事件
8. `attachStructuredContext(messages, attachments)` — 跨压缩保存上下文
9. 3 种压缩类型: auto / manual / reactive
10. 重试/恢复: 3 次流式重试 + 3 次 PTL 重试

## Task 2: Agent 生成工具 (P0)

**来源:** OpenHarness `tools/agent_tool.py`

- `AgentTool` — 生成子代理执行独立任务
- 支持 mode: local_agent / remote_agent
- 输入: task_description, mode, max_turns, model
- 返回: 子代理输出

## Task 3: SendMessage 工具 (P0)

**来源:** OpenHarness `tools/send_message_tool.py`

- `SendMessageTool` — 代理间消息传递
- 输入: message, recipient
- 支持 task_id 格式: name@team

## Task 4: 计划模式工具 (P1)

**来源:** OpenHarness `tools/enter_plan_mode_tool.py`, `exit_plan_mode_tool.py`

- `enterPlanMode()` — 切换到计划模式，阻止变更性工具
- `exitPlanMode()` — 恢复先前权限模式

## Task 5: 工作树工具 (P1)

**来源:** OpenHarness `tools/enter_worktree_tool.py`, `swarm/worktree.py`

- `createWorktree(branch, baseRef)` — 创建隔离 git worktree
- `removeWorktree(name)` — 清理工作树
