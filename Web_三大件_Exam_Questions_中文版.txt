第 1 题 — HTML 基本骨架（6 分）
--------------------------------------------------------------------------------

主题：HTML

请编写一个完整的 HTML5 页面，包含以下内容：

1. 正确的 doctype 声明
2. <html>、<head>、<body> 标签
3. 一个 <title>，内容为 "My First Webpage"
4. 一个主标题（<h1>），内容为 "Hello World!"
5. 一个段落（<p>），内容为 "This is my first webpage."

第 8 题 — 基本选择器（6 分）
--------------------------------------------------------------------------------

主题：CSS

给定以下 HTML：

  <h1>Welcome</h1>
  <p class="highlight">This is important.</p>
  <p>This is normal.</p>
  <div id="box">This is a box.</div>

编写 CSS 规则，实现以下效果：

1. 将 <h1> 的文字颜色设为红色
2. 将 class 为 "highlight" 的段落背景色设为黄色
3. 将 <div id="box"> 设置为 2px 实线蓝色边框，内边距 10px
4. 将所有 <p> 元素的字号设为 18px


--------------------------------------------------------------------------------
第 9 题 — 颜色和背景（6 分）
--------------------------------------------------------------------------------

主题：CSS

编写 CSS 来设置网页样式，要求如下：

1. 将整个页面（<body>）的背景色设为 #f5f5f5
2. 将所有 <h1> 元素的文字颜色设为深蓝色（#003366）

4. 将所有 <p> 元素的背景色设为白色，内边距为 10px

第 10 题 — 字体和文本（6 分）

第 11 题 — 盒模型（6 分）
--------------------------------------------------------------------------------

主题：CSS

给定以下 HTML：

  <div class="card">
    <h3>Card Title</h3>
    <p>This is the card content.</p>
  </div>

编写 CSS，实现以下效果：

1. 将 .card 的宽度设为 300px
2. 在卡片内部添加 20px 的内边距（padding）
3. 在卡片周围添加 1px 实线 #ccc 边框
4. 添加 margin: 50px auto 使其水平居中
5. 将背景色设为 #ffffff


--------------------------------------------------------------------------------
第 12 题 — Flexbox 基础（6 分）
--------------------------------------------------------------------------------

主题：CSS

创建一个包含 3 个盒子的 flexbox 容器。

1. 编写 HTML：一个 <div class="container"> 包含 3 个 <div class="box"> 元素
2. 编写 CSS，实现以下效果：
   - 使用 display: flex 将容器设为 flexbox
   - 盒子之间设置 20px 间距（gap）
   - 每个盒子宽度 100px、高度 100px、背景色为浅蓝色（lightblue）
   - 使用 justify-content: center 将盒子在容器内水平居中

第 13 题 — Flexbox 导航栏（6 分）
--------------------------------------------------------------------------------

主题：CSS

使用 Flexbox 创建一个简单的导航栏。

1. 编写 HTML：一个 <nav> 元素，内含一个无序列表（<ul>），包含 4 个链接：Home、About、Blog、Contact
2. 编写 CSS，实现以下效果：
   - 去除列表的项目符号（bullet points）
   - 对 <ul> 使用 display: flex，使列表项水平排列
   - 使用 justify-content: space-around 使列表项均匀分布
   - 给每个 <li> 添加一些内边距，背景色设为 #444
   - 链接文字设为白色，去掉下划线

第 14 题 — CSS 综合复习（6 分）
--------------------------------------------------------------------------------

主题：CSS

给定以下 HTML：

  <div class="profile-card">
    <h2>John Doe</h2>
    <p class="title">Web Developer</p>
    <p>I love building websites and learning new technologies.</p>
  </div>

编写 CSS，将其样式化为一张个人名片（profile card）：

1. 为 .profile-card 设置：宽度 350px、内边距 25px、外边距 30px auto、1px 实线 #ddd 边框、圆角（border-radius: 10px）
2. 将 <h2> 文字居中
3. 将 .title 文字设为灰色和斜体
4. 为卡片添加阴影：box-shadow: 2px 2px 10px rgba(0,0,0,0.1)
5. 将整个页面（<body>）的背景色设为 #f0f0f0

第 19 题 — DOM 操作（6 分）
--------------------------------------------------------------------------------

主题：JavaScript（DOM）

编写一个完整的 HTML 页面（包含内嵌 <script>），实现以下功能：

1. 显示一个标题："My Counter"
2. 显示一个 <p id="counter">，显示数字 0
3. 显示两个按钮："Increase"（增加）和 "Decrease"（减少）
4. 当用户点击 "Increase" 时，数字加 1
5. 当用户点击 "Decrease" 时，数字减 1

要求：
- 使用 document.getElementById() 获取元素
- 使用 addEventListener() 处理点击事件
- 使用变量记录计数值
- 每次计数变化时更新 <p> 中的文本

第 20 题 — 小项目：猜数字游戏（6 分）
--------------------------------------------------------------------------------

主题：HTML + CSS + JavaScript（综合）

在单个 HTML 文件中构建一个简单的猜数字游戏。

HTML（2 分）：
- 一个标题："Guess the Number!"
- 一个说明段落："I am thinking of a number between 1 and 10."
- 一个用于输入猜测值的文本输入框
- 一个显示 "Guess" 的按钮
- 一个 <p id="result">，用于显示结果信息

CSS（2 分）：
- 使用 Flexbox 将内容在 <body> 中居中
- 按钮背景色为蓝色，文字为白色，添加一些内边距
- 元素之间添加外边距以产生间距

JavaScript（2 分）：
- 页面加载时，使用 Math.random() 和 Math.floor() 生成 1 到 10 之间的随机数
- 点击按钮时：
  - 从输入框中读取用户的猜测值
  - 如果猜对了，显示："Correct! You guessed it!"
  - 如果猜高了，显示："Too high! Try again."
  - 如果猜低了，显示："Too low! Try again."
  - 如果输入为空，显示："Please enter a number!"
