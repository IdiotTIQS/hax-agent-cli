第 1 题 — 基本 HTML 骨架（6 分）
--------------------------------------------------------------------------------

题目：HTML

答案：

<!DOCTYPE html>
<html>
  <head>
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>

解释：
- 第 1 行：<!DOCTYPE html> 声明这是一个 HTML5 文档
- <html> 是页面的根元素
- <head> 包含元信息和页面标题
- <title> 设置浏览器标签页上显示的文本
- <body> 包含所有可见内容
- <h1> 是一级标题（最大的标题）
- <p> 是段落元素

第 3 题 — 链接和图片（6 分）
--------------------------------------------------------------------------------

题目：HTML

答案：

<!DOCTYPE html>
<html>
  <head>
    <title>Links and Images</title>
  </head>
  <body>

第 6 题 — 表单（6 分）
--------------------------------------------------------------------------------

题目：HTML

答案：

<!DOCTYPE html>
<html>
  <head>
    <title>Login Form</title>
  </head>
  <body>
    <h1>Login</h1>

    <form method="POST" action="/login">
      <label for="username">Username:</label>
      <input type="text" id="username" name="username">
      <br><br>

      <label for="password">Password:</label>
      <input type="password" id="password" name="password">
      <br><br>

      <input type="checkbox" id="remember" name="remember">
      <label for="remember">Remember me</label>
      <br><br>

      <input type="submit" value="Log In">
    </form>
  </body>
</html>

解释：
- <form> 包裹所有表单字段；method="POST" 隐藏发送数据，action="/login" 是服务器地址
- <label> 为每个输入框提供文本标签；"for" 属性链接到输入框的 "id"
- <input type="text"> 是单行文本框
- <input type="password"> 隐藏输入的字符
- <input type="checkbox"> 创建可勾选的框
- <input type="submit"> 创建提交表单数据的按钮

第 8 题 — 基础选择器（6 分）
--------------------------------------------------------------------------------

题目：CSS

答案：

/* 1. 将 <h1> 的文字颜色设为红色 */
h1 {
  color: red;
}

/* 2. 将 class 为 "highlight" 的段落背景色设为黄色 */

  background-color: yellow;

第 11 题 — 盒模型（6 分）
--------------------------------------------------------------------------------

题目：CSS

答案：

.card {
  width: 300px;
  padding: 20px;
  border: 1px solid #ccc;
  margin: 50px auto;
  background-color: #ffffff;
}

解释：
- width: 300px 设置卡片的**内容宽度**
- padding: 20px 在内容与边框之间添加空间（元素**内部**）
- border: 1px solid #ccc 在元素周围创建可见的轮廓线
- margin: 50px auto 在元素**外部**添加空间；"auto" 使其水平居中
- CSS 盒模型：从内到外依次为 → 内容(content) | 内边距(padding) | 边框(border) | 外边距(margin)

第 12 题 — Flexbox 基础（6 分）
--------------------------------------------------------------------------------

题目：CSS

答案：

HTML：

<div class="container">
  <div class="box">Box 1</div>
  <div class="box">Box 2</div>
  <div class="box">Box 3</div>
</div>

CSS：

.container {
  display: flex;
  gap: 20px;
  justify-content: center;
}

.box {
  width: 100px;
  height: 100px;
  background-color: lightblue;
}

解释：
- display: flex 将容器变为弹性盒子
- gap: 20px 在 flex 子项之间添加 20px 间距
- justify-content: center 将子项在容器中水平居中
- 每个 .box 是一个 flex 子项，有固定宽度和高度

题目：CSS

HTML：

<nav>

    <li><a href="#">About</a></li>
    <li><a href="#">Blog</a></li>

第 15 题 — 变量和数据类型（6 分）
--------------------------------------------------------------------------------

题目：JavaScript（基础）

答案：

// 1. 声明一个字符串变量 name
let name = "Alice";

// 2. 声明一个数字变量 age
let age = 20;

// 3. 声明一个布尔变量 isStudent
let isStudent = true;

// 4. 声明一个数组变量 hobbies
let hobbies = ["Reading", "Swimming", "Coding"];

// 5. 在控制台打印所有变量
console.log(name);      // 输出: Alice
console.log(age);       // 输出: 20

- 数字不使用引号
- 布尔值直接用 true/false（不加引号）
- 数组用方括号 [] 保存多个值

第 16 题 — 条件判断（if/else）（6 分）
--------------------------------------------------------------------------------

题目：JavaScript

答案：

// 声明一个分数变量
let score = 75;

// 使用 if/else if/else 判断等级
if (score >= 90) {
  console.log("Grade A");
} else if (score >= 80) {
  console.log("Grade B");
} else if (score >= 70) {
  console.log("Grade C");
} else if (score >= 60) {
  console.log("Grade D");
} else {
  console.log("Grade F");
}

// 因为 score = 75，所以输出为: Grade C

解释：
- if 检查第一个条件；如果为 true，执行该代码块
- else if 在前一个条件为 false 时检查下一个条件
- else 在所有条件都不为 true 时执行
- >= 表示"大于或等于"
- 条件按从上到下的顺序依次检查

第 17 题 — 循环（6 分）
--------------------------------------------------------------------------------

题目：JavaScript

答案：

// 1. 创建一个水果数组
let fruits = ["Apple", "Banana", "Cherry", "Orange", "Grape"];

// 2. 使用 for 循环打印每个水果
// 循环从 i=0 开始，当 i < fruits.length 时继续执行
for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

// 3. 使用 while 循环打印数字 1 到 5
let num = 1;
while (num <= 5) {
  console.log(num);
  num++; // 每次循环将 num 增加 1
}

解释：
- for 循环：(初始值; 条件; 增量) —— 在已知循环次数时使用
  - i = 0 从第一个索引开始，i++ 每次循环增加 1
  - fruits[i] 通过索引访问每个元素
- while 循环：只要条件为 true 就一直运行
  - 必须手动更新计数器（num++)，否则会导致死循环

第 18 题 — 函数（6 分）
--------------------------------------------------------------------------------

题目：JavaScript

答案：

// 1. 声明一个名为 greet 的函数
// 它接收一个 name 参数，返回问候语字符串
function greet(name) {
  return "Hello, " + name + "! Nice to meet you!";
}

// 2. 声明一个名为 add 的函数
// 它接收两个数字，返回它们的和
function add(a, b) {
  return a + b;
}

// 3. 调用两个函数并打印结果
console.log(greet("Bob"));       // 输出: Hello, Bob! Nice to meet you!
console.log(add(5, 3));          // 输出: 8

解释：
- function 是声明函数的关键字
- 参数 (name, a, b) 是函数接收的输入
- return 将值返回给函数被调用的地方
- + 运算符既可以拼接字符串，也可以对数字做加法

第 19 题 — DOM 操作（6 分）
--------------------------------------------------------------------------------

题目：JavaScript（DOM）

答案：

<!DOCTYPE html>
<html>
<head>
  <title>My Counter</title>
</head>
<body>

  <h1>My Counter</h1>
  <p id="counter">0</p>
  <button id="increase">Increase</button>
  <button id="decrease">Decrease</button>

  <script>
    // 获取 HTML 元素的引用
    let counterDisplay = document.getElementById("counter");
    let increaseBtn = document.getElementById("increase");
    let decreaseBtn = document.getElementById("decrease");

    // 记录当前计数值
    let count = 0;

    // 给 Increase 按钮添加点击事件
    increaseBtn.addEventListener("click", function() {
      count = count + 1;              // 计数值增加 1
      counterDisplay.innerText = count; // 更新显示
    });

    // 给 Decrease 按钮添加点击事件
    decreaseBtn.addEventListener("click", function() {
      count = count - 1;              // 计数值减少 1
      counterDisplay.innerText = count; // 更新显示
    });
  </script>

</body>
</html>

解释：
- document.getElementById("id") 通过 id 属性查找 HTML 元素
- addEventListener("click", function) 在元素被点击时运行代码
- .innerText 改变元素内显示的可见文本
- count 变量存储当前数值；每次点击增加或减少 1

第 20 题 — 小项目：猜数字游戏（6 分）
--------------------------------------------------------------------------------

题目：HTML + CSS + JavaScript（综合）

答案：

<!DOCTYPE html>
<html>
<head>
  <title>Number Guessing Game</title>
  <style>
    /* 使用 Flexbox 将所有内容居中 */
    body {
      display: flex;
      flex-direction: column;
      align-items: center;
      justify-content: center;
      min-height: 100vh;
      font-family: Arial, sans-serif;
    }

    /* 按钮样式 */
    button {
      background-color: blue;
      color: white;
      padding: 10px 20px;
      border: none;
      border-radius: 5px;
      cursor: pointer;
    }

    /* 元素之间添加间距 */
    input, button, p {
      margin: 10px;
    }

    input {
      padding: 8px;
      font-size: 16px;
    }

    p {
      font-size: 18px;
    }
  </style>
</head>
<body>

  <h1>Guess the Number!</h1>
  <p>I am thinking of a number between 1 and 10.</p>
  <input type="text" id="guessInput" placeholder="Enter your guess">
  <button id="guessBtn">Guess</button>
  <p id="result"></p>

  <script>
    // 页面加载时生成一个 1 到 10 之间的随机数
    let secretNumber = Math.floor(Math.random() * 10) + 1;

    // 获取 HTML 元素的引用
    let guessInput = document.getElementById("guessInput");
    let guessBtn = document.getElementById("guessBtn");
    let resultDisplay = document.getElementById("result");

    // 给 Guess 按钮添加点击事件
    guessBtn.addEventListener("click", function() {
      let userGuess = guessInput.value;  // 读取输入框的值

      // 检查输入是否为空
      if (userGuess === "") {
        resultDisplay.innerText = "Please enter a number!";
        return;  // 停止执行后续代码
      }

      // 将输入的字符串转换为数字
      userGuess = Number(userGuess);

      // 将用户的猜测与秘密数字进行比较
      if (userGuess === secretNumber) {
        resultDisplay.innerText = "Correct! You guessed it!";
      } else if (userGuess > secretNumber) {
        resultDisplay.innerText = "Too high! Try again.";
      } else {
        resultDisplay.innerText = "Too low! Try again.";
      }
    });
  </script>

</body>
</html>

解释：
- Math.random() 生成一个 0 到 1 之间的随机小数
- Math.floor() 向下取整到最近的整数
- Math.floor(Math.random() * 10) + 1 得到 1 到 10 之间的随机整数
- Number() 将字符串转换为数字，以便进行比较
- === 运算符检查严格相等（值和类型都必须相同）
